## Decisões técnicas

- **Next.js 16** (não 15 — versão atual ao criar o projeto)
- **`proxy.ts`** em vez de `middleware.ts` — breaking change do Next.js 16
- Guard no proxy: quando Supabase não está configurado, passa sem errar
- Rotas de dashboard dentro de `app/(dashboard)/dashboard/` para gerar URLs `/dashboard/*`
- Tailwind v3 mantido (downgrade manual de v4)

## Problemas resolvidos

- Tailwind v4 instalado por padrão → downgrade para v3
- `tw-animate-css` (v4) → substituído por `tailwindcss-animate` (v3)
- `class-variance-authority` ausente → instalada manualmente
- `@import` Google Fonts deve vir ANTES dos `@tailwind` directives no CSS
- Next.js 16: `middleware.ts` → renomeado para `proxy.ts` com export `proxy`
- Rotas sem prefixo `/dashboard` → movidas para subpasta correta
