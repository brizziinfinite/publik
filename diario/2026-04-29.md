## Decisões técnicas

- **`distance: 8` no PointerSensor**: clique simples ainda abre PostDialog; drag ativa após 8px
- **service_role key no pg_cron SQL**: único lugar seguro para chamadas internas do banco
- **framer-motion `mode="wait"`**: evita overlap ao trocar páginas
- **`user_metadata` para preferências**: sem nova tabela para MVP
- **Toggles sem lib**: 3 switches Tailwind puros são mais leves que `@radix-ui/react-switch`
- **`Suspense` para `useSearchParams`**: isolado em sub-componente `OAuthToastHandler` para evitar erro de build do Next.js
- **Instagram usa OAuth do Meta**: mesmo flow do Facebook — o callback detecta automaticamente se há conta Instagram Business vinculada à página

## Problemas resolvidos

- **`lucide-react` sem Instagram/Facebook**: versão instalada não exporta esses ícones — substituídos por SVG inline
- **`useSearchParams` sem Suspense**: Next.js 16 exige Suspense boundary; resolvido com sub-componente isolado
- **`ALTER DATABASE` sem permissão**: Supabase Free não permite; service_role key direto no SQL do cron
- **tsconfig incluindo Deno**: adicionado `"supabase/functions"` ao `exclude`
- **Branches fora de sync**: main estava no scaffold; merge de feat branch antes de implementar
