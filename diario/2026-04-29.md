## Decisões técnicas

- **`distance: 8` no PointerSensor**: garante que um clique simples no chip ainda abre o PostDialog; o drag só ativa após 8px de movimento
- **service_role key no pg_cron SQL**: único lugar seguro para chamadas internas do banco; a key não fica no frontend nem no código-fonte versionado (está no banco criptografado)
- **framer-motion `mode="wait"`**: espera a página atual sair antes de animar a entrada da nova — evita overlap de conteúdos
- **`user_metadata` para preferências**: evita criar nova tabela; simples para MVP; pode migrar para tabela própria quando houver mais campos
- **Toggles sem lib**: 3 switches simples com Tailwind são mais leves que importar `@radix-ui/react-switch` só pra isso

## Problemas resolvidos

- **Branches fora de sync**: `main` estava no scaffold, features nas branches de feat. Resolvido com merge na branch de trabalho antes de implementar
- **tsconfig incluindo Deno**: Edge Functions usam `Deno.*` e imports `https://esm.sh/*` — TypeScript do Next.js não reconhecia. Adicionado `"supabase/functions"` ao `exclude` do `tsconfig.json`
- **`ALTER DATABASE` sem permissão**: Supabase Free não permite `ALTER DATABASE SET`. Resolvido colocando a service_role key diretamente no SQL do cron job
