## Decisões técnicas

- `runtime = 'nodejs'` na API Route — necessário para `fs/promises` (leitura de fontes)
- `@vercel/og` (não `next/og`) importado diretamente — versão 0.11.1 compatível com Next 16
- `safe_zone as unknown as RenderFormatSafeZone` — jsonb do Supabase é tipado como `Json` opaco
- Template usa apenas `display: flex` e inline styles (restrição Satori)
- `reloadRef` pattern para chamar `load()` de dentro de handler sem adicionar como dep de useEffect
- Fontes em cache `Map<string, ArrayBuffer>` — evita leitura repetida em re-renders
